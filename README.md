# myapp
Prompt 1: Estructura Global y Pantalla de Inicio
"Configura la estructura base de una app en Flutter para mi negocio Netflix EALG 0624 6 I. Utiliza un Scaffold con un BottomNavigationBar de 5 opciones. La Pantalla de Inicio debe tener un AppBar con el nombre del negocio en rojo. En el centro, coloca una imagen de la web que sea el logo horizontal de Netflix (PNG transparente) con un tamaño de 300x150. Añade un efecto de sombra roja suave detrás del logo y el texto 'BIENVENIDO' debajo en negrita."
A new Flutter project.

Prompt 2: Pantalla de Cine (Películas)
"Diseña la segunda sección de la app llamada Cine. El encabezado debe decir Netflix EALG 0624 6 I. En el centro de la pantalla, muestra un poster vertical de una película de tendencia (usa una URL de imagen real como Mad Max o similar) con un tamaño de 250x360 y bordes redondeados (BorderRadius.circular(15)). Debajo de la imagen, añade el subtítulo 'Tendencias en Cine' con una separación de 25 píxeles."

Prompt 3: Pantalla de Series TV
"Crea la tercera sección dedicada a Series. Mantén el AppBar con el nombre del negocio. Para esta pantalla, usa una imagen de la web que sea un poster vertical de una serie icónica (ejemplo: Stranger Things) con las mismas dimensiones que la de cine (250x360). Asegúrate de incluir un errorBuilder para que, si el link falla, se muestre un icono de Icons.play_circle_fill en color rojo. El subtítulo debe ser 'Lo más visto'."

Prompt 4: Pantalla de Perfil de Usuario
"Configura la cuarta pantalla llamada Perfil. En el centro, muestra el avatar clásico cuadrado de Netflix (color azul) obtenido desde la web. La imagen debe tener bordes redondeados de 10 píxeles. Debajo de la foto, coloca en letras grandes y blancas el nombre del administrador: 'EALG 0624 6 I', y en una línea inferior con color gris, el texto 'Cuenta Premium'."

Prompt 5: Pantalla de Ajustes (Panel de Control)
"Diseña la última sección llamada Ajustes. A diferencia de las anteriores, esta pantalla debe ser minimalista: sin listas ni imágenes de red. Coloca en el centro un único icono grande (size: 160) de configuración como Icons.manage_accounts o Icons.tune en un color blanco sutil o gris oscuro. Debajo del icono, añade el texto 'Preferencias del Sistema' con un espaciado de letras (letter-spacing) de 1.5 para un toque elegante."
